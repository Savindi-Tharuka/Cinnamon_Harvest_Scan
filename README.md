# Cinnamon_Harvest_Scan
# Expo React Native App

Cross-platform mobile app (Android + iOS) for cinnamon stem capture, analysis, and history tracking,month prediction using react native with expo go app for frontend

## Implemented flow
1. Home page with:
   - Upload Image
   - Scan Stem (routes to Tips)
2. Tips page with guided capture instructions + Scan Now button.
3. Camera page with in-app camera, framing guidance, and photo capture.
4. Preview page with Retake / Continue.
5. Result page with skeleton loading, then result details and Save / Discard actions.
6. Display the month prediction for Immatured plants stems using ruler based method
7. History page with:
   - Status filter
   - Analyzed date range filter
   - Infinite scroll pagination
   - Open result and Delete actions per record
